## Overview

EpiphAI is a no-code platform by Digital Alpha that enables businesses to build and deploy AI agents for automation without requiring programming skills. The platform streamlines operations by automating repetitive tasks, increasing productivity, and letting teams focus on higher-value work.

**Platform**: [dev.epiphaiplatform.com](https://dev.epiphaiplatform.com)

This project was developed during my internship at Digital Alpha, where I contributed to building core features and infrastructure.

## Platform Features

- **No-Code Agent Builder**: Create AI agents customized for business processes through an intuitive interface
- **Pre-Built Templates**: Quick-start templates for common business scenarios
- **Dynamic Triggers**: Event-based workflow automation for responsive operations
- **Pre-Trained Agents**: Industry-specific agents ready for instant integration
- **Open Source Flexibility**: Advanced custom agent development using open-source tools
- **Seamless Integration**: Connect with existing tools (CRM, cloud storage, databases) without disrupting workflows
- **LLM Flexibility**: Choose the most suitable large language model for varied needs

## Technologies Used

- **Backend**: Django, Django REST Framework, Celery
- **Frontend**: React.js, Markdown rendering
- **Cloud Services**: AWS (SQS, SES, Lambda)
- **Database**: PostgreSQL, Redis
- **Message Queue**: Celery with Redis broker
- **API Documentation**: OpenAPI/Swagger
- **Real-time Features**: WebSocket streaming
