## Tips
1. Use browser screenshot tools (Cmd+Shift+4 on Mac, Win+Shift+S on Windows)
2. For full-page screenshots, use browser extensions like "Full Page Screen Capture"
3. Compress images before adding (use tinypng.com or squoosh.app)
4. Recommended dimensions: 1600x900px for project screenshots, 800x400px for logos
